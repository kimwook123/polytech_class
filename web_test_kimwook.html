사람들은 왜 강아지에 열광할까?

  1. 굉장히 귀엽다!
    2. 이것은 아무도 부정할 수 없는, 필연적 사실이다.

  2. 사람을 정말정말 좋아한다!
    3. 사람을 너무너무 좋아해서, 곁에서 떠나질 않고 애교를 부린다.(엄청나게!)

  3. 사람을 도울 줄 안다!
    4. 마찬가지로 사람을 너무너무 좋아해서, 자기가 되는 일이라면 뭔가 도와주고 싶어한다!

  4. 교육을 시키면 못하는 게 없다?!

    5. 보더콜리 또는 리트리버와 같은 똑똑한 강아지들은

    양치기, 군견, 마약탐지견,시각장애인 안내견

    과 같은 전문적인 일을 교육받고 실제 업무를 굉장히 성공적으로 수행한다!!!
